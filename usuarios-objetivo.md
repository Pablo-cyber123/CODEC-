# Sistema de almacenamiento de usuarios
 
 Se almacenaran los nombres y datos de los usuarios en un archivo .json ya que se tratan de datos que se agregan en la pagina

 se implementará un sistema de registro y almacenamiento, serán el iniciar sesion, registro y el sistema de almacenamiento de usuarios
